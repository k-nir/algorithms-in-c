Big-Oh Notation

大O表示法

简介

大O表示法(o-notation或big-Oh notation)是一种数学记号，用于描述函数在趋近某个极限时具有的行为。 (即函数在其参数趋近特定值的情况下值增长/减小的速率，该特定值通常是无穷大或无穷小)

大O表示法和一些类似的表示法(小o/大Theta/大Omega/小Omega表示法等)一起组成一个大类， 称为巴赫曼-朗道表示法(Bachmann-Landau notation)或渐进表示法(asymptotic notation)。

正式定义

大O表示法有两种正式定义， 一种定义默认被描述函数的定义域是无界集， 另一种定义将被描述函数的定义域限定在特定值的周围。

设有值为实数或复数的函数 g(N)，值为实数的函数 f(N)，

在第一种定义中， g(N) 和 f(N) 的定义域是正实数域中的某个无界集(unbounded set)(即右侧为正无穷的区间)， 此时，若存在正实数 c0 和实数 N0，使得:

|g(N)| ⩽ c0⋅f(N) for all N ⩾ N0

则可以说:

g(N) = O(f(N)) as N → ∞

上述表达式可以简写为:

g(N) = O(f(N))

在第二种定义中， g(N) 和 f(N) 的定义域是实数 a 附近的区域， 存在一个正实数 d，使得 0 < |N − a| < d， 此时，若存在正实数 c0 和实数 N0，使得:

|g(N)| ⩽ c0⋅f(N) for all N with 0 < |N − a| < d

则可以说:

g(N) = O(f(N)) as N → a

第二种定义中的 a 通常是0。

应用

大O表示法主要有两种用法:

  • 描述函数趋近无穷大时的行为(称为无穷大渐进)
  • 描述函数趋近无穷小时的行为(称为无穷小渐进)

无穷大渐进

该用法用大O表示法概括复杂函数 f(N) 在 N → ∞ 过程中的行为(增长速率)。

以函数 T(n) = 4n2 − 2n + 2 为例，用大O表示法可将其表示为: T(n) = O(n2)。

推导过程

在 T(n) 中，n 的值越大( n > 1 的情况下)，4n2 项对 T(n) 的值的影响越大。 例如，当 n = 500 时，4n2 比 2n 大1000倍，使得 2n 项对 T(n) 的值几乎没有影响。 因此，T(n) 的增长速率由 4n2 决定，即在 n 越大的情况下， T(N) 的增长速率越接近 4n2。

既然 T(n) 的增长速率会趋近 4n2，那么在两者足够接近的情况下(即 n 超过某个 n0 以后), 一定存在一个 c0⋅n2 (c0 为常量)，使 T(n) ⩽ c0⋅n2 (例如: c0 = 5 ,  n0 = 3)。 这意味着在区间 [N0, ∞) 内，c0⋅n2 的图形都在 T(n) 之上， 即 c0⋅n2 能概括 T(n) 在 [N0, ∞) 区间内的行为。

因此，我们可以说 T(n) = O(n2)。

对次要项的省略

O(n2) 与 T(n) 相比，省略了二次项以外的所有项。 这样的省略之所以可取的原因，已经在推导过程中提到，就是因为与二次项相比，次要项对函数的增长的贡献微不足道。

对常量系数的省略

O(n2) 与 c0⋅n2 相比，省略了常量系数。 之所以可以省略常量系数，是因为大O表示法关注的焦点是函数的增长速率， c0⋅n2 中决定增长速率(函数图像形状)的是 n2， c0 只决定函数图像在纵轴上的位置(不影响形状)。

从另一个视角也可以看出常量系数的次要性。 比较 c0⋅n2 与 U(n) = n3， 无论 c0 有多大，c0⋅n2 的函数图像都会在某个点被 U(n) 的函数图像超越。

无穷小渐进

该用法用大O表示法概括复杂函数 f(N) 在 N → 0 过程中的行为(增长速率)。

以指数函数 ex 为例，用大O表示法可将其表示为: ex = 1 + x + O(x2)  as x → 0。

推导过程

ex 可展开为 1 + x + (x2)/(2!) + (x3)/(3!) + (x4)/(4!) + ⋯， 在展开式中，x 越接近0， 指数项对 ex 的值的影响越小。 例如，当 x = 0.05 时，1 + x = 1.05，指数项的值为 ex − (1 + x) = 0.00127…，对 ex 的结果影响非常小。
